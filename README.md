# Z3n Marketplace

A modern digital marketplace built with vanilla HTML, CSS, JavaScript, and Supabase.

## Getting Started

1. Copy `.env.example` to `.env` and add your Supabase anon key.
2. Serve the `z3n` folder with any static server (e.g. `npx serve` or VS Code Live Server).
3. Open `src/pages/index.html` in your browser.
